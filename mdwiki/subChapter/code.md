# Code einbinden

## Beispiel mit Javascript

Folgendes:

    ```javascript
    var hello = function () {
        // say hello
        alert('Hello world!');
    }
    ```

generiert folgendes:

```javascript
var hello = function () {
    // say hello
    alert('Hello world!');
}
```
